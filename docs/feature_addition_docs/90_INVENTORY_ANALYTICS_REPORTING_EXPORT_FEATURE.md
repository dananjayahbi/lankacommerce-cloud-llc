## Backend API Requirements

### GET /api/inventory/reports/saved-reports/
Get list of saved reports

**Query Parameters:**
- `limit` (optional): Pagination limit
- `offset` (optional): Pagination offset
- `sort_by` (optional): Sorting field
- `filter_type` (optional): Filter by report type
- `date_range` (optional): Filter by date range

**Response:**
```
{
  count: number,
  results: [
    {
      id: string,
      name: string,
      type: string,
      generated_at: string (ISO),
      generated_by: string,
      format: string,
      file_size: number,
      status: string
    }
  ]
}
```

### GET /api/inventory/reports/saved-reports/{id}/
Get report details

**Response:**
```
{
  id: string,
  name: string,
  type: string,
  generated_at: string (ISO),
  file_url: string,
  parameters: object,
  metadata: object
}
```

### GET /api/inventory/reports/saved-reports/{id}/download/
Download report file

**Query Parameters:**
- `format` (optional): Format conversion if needed

**Response:** File download (binary)

### POST /api/inventory/reports/saved-reports/{id}/email/
Email report

**Request Body:**
```
{
  recipients: string[],
  subject: string,
  message: string (optional)
}
```

**Response:**
```
{
  success: boolean
}
```

### DELETE /api/inventory/reports/saved-reports/{id}/
Delete report

**Response:**
```
{
  success: boolean
}
```

### POST /api/inventory/reports/generate/
Generate custom report

**Request Body:**
```
{
  report_type: string,
  start_date: string (ISO),
  end_date: string (ISO),
  format: string,
  warehouse_id: string (optional),
  filters: object (optional)
}
```

**Response:**
```
{
  id: string,
  status: string,
  estimated_time_ms: number,
  file_url: string (optional, when ready)
}
```

### GET /api/inventory/reports/{id}/status/
Check generation status

**Response:**
```
{
  id: string,
  status: string,
  progress_percent: number,
  error: string (optional, if failed)
}
```

### GET /api/inventory/reports/scheduled-reports/
Get scheduled reports (if enabled)

**Response:**
```
[
  {
    id: string,
    report_type: string,
    schedule: string (cron),
    next_run: string (ISO),
    last_run: string (ISO),
    status: string
  }
]
```

### POST /api/inventory/reports/scheduled-reports/
Create scheduled report

**Request Body:**
```
{
  report_type: string,
  schedule: string (cron),
  email_recipients: string[],
  parameters: object
}
```

**Response:**
```
{
  id: string,
  schedule: string,
  next_run: string (ISO)
}
```

### PATCH /api/inventory/reports/scheduled-reports/{id}/
Update scheduled report

**Request Body:**
```
{
  schedule: string (optional, cron),
  email_recipients: string[] (optional),
  parameters: object (optional)
}
```

**Response:** Updated scheduled report object

### DELETE /api/inventory/reports/scheduled-reports/{id}/
Delete scheduled report

**Response:**
```
{
  success: boolean
}
```

## Database Requirements

### Data Models

#### SavedReport Model (Enhancement if Needed)
- `id`: Primary key
- `tenant_id`: Multi-tenant isolation
- `name`: Report name
- `report_type`: Type of report
- `parameters`: JSON field for filter parameters
- `generated_at`: Generation timestamp
- `file_path`: Path to generated file
- `file_format`: Export format (PDF, Excel, CSV)
- `file_size`: File size in bytes
- `status`: Generation status (pending, completed, failed)
- `generated_by`: User who generated report

#### ScheduledReport Model (if Feature Enabled)
- `id`: Primary key
- `tenant_id`: Multi-tenant isolation
- `report_type`: Type of report
- `schedule`: Cron expression
- `email_recipients`: Array of email addresses
- `parameters`: JSON field for filter parameters
- `next_run`: Next scheduled run datetime
- `last_run`: Last execution datetime
- `status`: Active/inactive

### Indexes Required
- `(tenant_id, generated_at DESC)`
- `(report_type, generated_at)`
- `(status, created_at)`
