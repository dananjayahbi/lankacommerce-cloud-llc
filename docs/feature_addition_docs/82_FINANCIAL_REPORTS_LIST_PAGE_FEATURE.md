# Financial Reports List Page Feature Specification

## Executive Summary

Financial Reports list and management interface enabling accounting teams to browse, generate, schedule, and manage financial reports including P&L statements, balance sheets, cash flow statements, trial balance, and custom financial analysis reports with caching, scheduling, and export capabilities.

## Current State Analysis

### EXISTING INFRASTRUCTURE:
- Reports app infrastructure (backend/apps/reports/)
- Date range selection components
- Export functionality framework (CSV, PDF support)
- Multi-tenant database
- User permissions system
- Audit logging capability

### MISSING (Partially implemented or new features):
- P&L Statement generation (requires Chart of Accounts, Journal Entries)
- Balance Sheet generation (requires GL account balances)
- Cash Flow Statement generation (requires transaction classification)
- Trial Balance generation (requires JE posting and account balances)
- General Ledger report (by account)
- Account Statement report (single account transactions)
- Financial Ratio Analysis (requires accounting data)
- Report templates storage
- Scheduled report execution
- Report history/archive
- Report caching for performance
- Report comparison (period-over-period)
- Report filtering options (department, cost center)
- Report customization (column selection, sorting)
- Report preview before generation
- Report notifications (email delivery)
- Report API endpoints
- Report generation services
- Report export services (PDF, Excel, CSV)
- Report scheduling service
- Frontend reports list page
- Report generation workflow UI
- Report viewer/display component
- Report filtering UI
- Frontend components not fully implemented

## Frontend Features

### Reports List Section:
- **Available reports display** (grid or list):
  - Report name (clickable)
  - Description
  - Report type indicator (Profit & Loss, Balance Sheet, Cash Flow, Trial Balance, General Ledger, Account Statement, Financial Ratios)
  - Last generated date
  - Generate button
  - View button (if recently generated)

- **Report categories** (collapsible sections):
  - Financial Statements (P&L, Balance Sheet, Cash Flow, Trial Balance)
  - Detailed Reports (General Ledger, Account Statement, Account Listing)
  - Analysis Reports (Financial Ratios, Variance Analysis)

- **Search bar** (search by report name)
- **Filter by report type**

### Recent Reports History Section:
- **Recently generated reports table**:
  - Columns: Report Name, Generated Date, Generated By, Format, Size, Actions
  - Sort by date descending
  - Limit to 10-20 recent
  - View button (opens report)
  - Download button
  - Delete button (archive old reports)
  - Rerun button (regenerates with same parameters)

- **Export recent reports list** (CSV)
- **Clear history button**
- **Pagination** if many recent reports

### Scheduled Reports Section (if feature enabled):
- **Scheduled reports table**:
  - Columns: Report Name, Schedule, Next Run, Status, Actions
  - Edit button (modify schedule)
  - Delete button (remove schedule)
  - View last result button
  - Enable/Disable toggle

- **Create new schedule button**
- **Add email recipients section**

### Report Templates Section (optional, if feature enabled):
- **Saved templates list**:
  - Template name
  - Description
  - Based on report
  - Last used
  - Use button
  - Delete button

- **Create template button**

### Quick Action Buttons:
- Generate P&L button
- Generate Balance Sheet button
- Generate Trial Balance button
- Generate Cash Flow button
- More reports link (opens report selection)

### Period Selector (prominent):
- Start date picker
- End date picker
- Preset options (This Month, Last Month, This Quarter, This Year, Last Year, YTD)
- Compare with period option (for comparison reports)

### Filter Options (collapsible):
- Department filter (multi-select)
- Cost center filter (multi-select)
- Account type filter (for filtered reports)
- Status filter (draft, final)
- Clear filters button

### Statistics Section (optional):
- Reports generated (this month)
- Total reports available
- Storage used for reports

### States:
- Loading state (skeleton loader)
- Empty state (if no reports, with "Generate Report" prompt)
- Error state (error message display)

## Backend API Requirements

### Endpoints:

**GET /api/accounting/reports/available-reports/**
- Get list of available reports
- Response: `[{ id, name, description, type, parameters: [...] }]`

**GET /api/accounting/reports/recent/**
- Get recently generated reports
- Query params: limit (default 10)
- Response: `[{ id, name, generated_at, generated_by, format, file_path, size }]`

**POST /api/accounting/reports/generate/**
- Generate a financial report
- Request body: `{ report_type, period_start, period_end, format (pdf/excel/csv), filters: {...} }`
- Response: `{ id, report_type, status, file_url, generated_at }`

**GET /api/accounting/reports/{id}/**
- Retrieve report details and file
- Response: file download or `{ id, type, status, generated_at, parameters, file_url }`

**GET /api/accounting/reports/scheduled/**
- Get scheduled reports (if enabled)
- Response: `[{ id, report_type, schedule, next_run, last_run, status }]`

**POST /api/accounting/reports/scheduled/**
- Create scheduled report

**PATCH /api/accounting/reports/scheduled/{id}/**
- Update scheduled report

**DELETE /api/accounting/reports/scheduled/{id}/**
- Delete scheduled report

**POST /api/accounting/reports/templates/**
- Save report as template
- Request body: `{ name, description, report_type, parameters }`

**GET /api/accounting/reports/templates/**
- Get saved templates

**GET /api/accounting/reports/download/{id}/**
- Download report file
- Query params: format (pdf/excel/csv)

## Validation & Edge Cases

- Period dates must be valid (start <= end, end <= today)
- Report generation requires accounting data to exist
- P&L and Balance Sheet require Chart of Accounts
- Trial Balance requires Journal Entries
- Cash Flow requires transaction classification
- Permissions required to view reports
- Multi-tenant data isolation
- Large reports may timeout (chunking/pagination needed)
- Report file cleanup (old files archived)
- Concurrent report generation handling
- Account balances calculation accuracy
- Period-end adjustments consideration
